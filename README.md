# 🍔 FoodieHub

FoodieHub is a food ordering web application inspired by Zomato and Swiggy.

## 🚀 Technologies Used

- Java
- JDBC
- Servlets
- JSP
- MySQL
- HTML
- CSS
- Apache Tomcat
- Maven
- Git & GitHub

## ✨ Features Completed

### User Authentication
- User Registration
- User Login
- User Logout
- Session Management

## 🚧 Features Under Development

- Restaurant Listing
- Restaurant Menu
- Cart
- Checkout
- Orders
- Reviews & Ratings
- Admin Dashboard
